# Smart Vehicle Safety System using AT89S52

A vehicle safety system developed using the AT89S52 microcontroller.

## Features
- Automatic headlight dimming using LDR
- Obstacle detection using HC-SR04
- Automatic braking
- PWM-based LED control
- DC motor control

## Technologies
- Embedded C
- AT89S52
- Keil uVision
